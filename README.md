# Amazon-Scraper
Given a search term or terms, it will return scraped data from a product's Amazon page.

# Scraped Data (for books)
- Title
- Author/Creator
- Reviews
- Price
- Paperback/hardback Pages
- Publisher
- Language
- ISBN-10
- ISBN-13
- Product Dimensions
- Shipping Weight
- Average Customer Review
- Amazon Best Sellers Rank
- Top User Review
- Description
